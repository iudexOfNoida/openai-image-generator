# OpenAI based image generator 
MERN stack application that can create images from prompt. You can share posts in community and also search for created posts and download them!
Technologies used are:
-  Node.js
-  OpenAi API
-  MongoDB
-  React
-  Cloudinary
-  Express
-  JavaScript
